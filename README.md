# PWA Dashboard

A Progressive Web App with Todo List and DevOps Dashboard.

## Features
- ✅ Works offline
- ✅ Installable on any device
- ✅ Dark/Light mode
- ✅ Todo list with localStorage persistence
- ✅ Real-time system monitoring
